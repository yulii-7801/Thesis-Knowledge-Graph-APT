# 面向盲审的APT知识图谱双空间表示学习实验重构

## 1. 项目简介
本项目为本人的硕士毕业设计完整归档资料。主要包含面向 APT 领域知识图谱的双空间表示学习模型的源代码、所用数据集、实验记录及最终毕业答辩相关文档。

## 2. 目录结构说明
* `01_毕业论文终稿与答辩ppt/`：存放最终版盲审论文 PDF 及答辩汇报 PPT。
* `02_代码/`：包含图神经网络等核心模型的网络定义及训练脚本。
* `03_数据集/`：存放实验所用的 APT 实体关系网络数据及预处理脚本。
* `04_实验数据/`：包含模型训练日志、保存的最优权重文件以及最终测试指标数据。。
* `05_论文参考文献pdf/`：毕业论文中引用的核心参考文献存档。

## 3. 环境依赖与运行指南 (快速启动)
环境要求：
* Python >= 3.7
* PyTorch >= 1.7.0 (若依赖特定 CUDA 版本，请自行匹配)
* NumPy
* SciPy
* tqdm (用于控制台进度条)

运行步骤：
1. 配置好依赖环境。
2. 进入 `02_代码` 目录。
3. 运行主程序：`main-backup.py`（替换为你的实际运行命令）。
